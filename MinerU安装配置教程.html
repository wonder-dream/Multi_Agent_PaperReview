MinerU 安装配置教程

1. 安装 Node.js

MinerU 通过 npm 安装，需要先装 Node.js。

去 https://nodejs.org 下载 LTS 版本，安装后验证：

node --version
npm --version

2. 安装 MinerU CLI

npm install -g mineru-open-api

验证：

mineru-open-api version

3. 获取 API Token（可选，flash-extract 不需要）

打开 https://mineru.net/apiManage/token，注册/登录后复制 Token。

mineru-open-api auth

按提示粘贴 Token 即可，保存后在 ~/.mineru/config.yaml。

4. 两种提取模式

模式 命令 Token 限制
快速 flash-extract 不需要 10MB / 20页
高精度 extract 需要 更高 
# 快速提取
mineru-open-api flash-extract file.pdf -o ./out/

# 高精度提取（支持 md/html/latex/docx/json）
mineru-open-api extract file.pdf -o ./out/ -f md

支持的输入格式：PDF、图片（png/jpg/webp等）、Word、PPT、Excel、URL。

5. 日常使用

mineru-open-api flash-extract 文件.pdf -o ./out/

mineru-open-api extract 文件.pdf -o ./out/ -f md

mineru-open-api extract 文件.pdf -f html

mineru-open-api extract 图片.png -f md

mineru-open-api extract *.pdf -f md
